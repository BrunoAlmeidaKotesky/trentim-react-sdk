# Measurement

Decorate a method with this decorator to measure the execution time of the method, the result will be printed on browser console.

* Parameters:

<TypesTable rows={[
    {
        Id: 1,
        name: 'disable',
        type: 'boolean',
        description: 'If true, the measurement is disabled.',
        default: 'false',
        required: false
    },
    {
        Id: 2,
        name: 'logCss',
        type: 'string',
        description: 'Custom `console.log` css if desired.',
        default: '`color: white; background-color: #3e88ef`',
        required: false
    }
]} />

* Example:

```ts dark
class Example {
    @Measure()
    public longLog(): number {
        let finalValue = 0;
        for (let i = 0; i < 100000000; i++) {
            finalValue += i;
        }
        return finalValue;
    }

    @Measure(false, 'color: white; background-color: rgb(132, 229, 217)')
    public logWithCss() {
        let finalValue = 0;
        for (let i = 0; i < 100000000; i++) {
            finalValue += i;
        }
        return finalValue;
    }
}
```
