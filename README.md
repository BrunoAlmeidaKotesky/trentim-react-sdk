## Trentim-React-SDK
The purpose of this library is to provide an easy way to use common UI Components and Utilities that we often use at Trentim. 
This is an Alpha version, which everything still can be changed.

### Components
* Datatable

### Utils
* CacheHandler
* FileUtils
* SPFx WebpartAddons

### React Hooks
* useDebounce
* useEscape
* useOuterClick
* usePrevious
* useWindowSize
